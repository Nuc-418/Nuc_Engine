# Controls & Keybindings

All gameplay keys go through [[Input|named actions]]; these are the default
bindings.

## Everywhere (engine defaults)

| Key | Action |
|---|---|
| W / S | MoveForward axis |
| D / A | MoveRight axis |
| Space / LCtrl | MoveUp axis |
| LShift | Sprint (2× speed) |
| Esc | Exit (closes the game; stops Play in the editor) |
| Mouse | Look (while flying / playing; cursor recentered) |

## Editor viewport

| Input | Effect |
|---|---|
| RMB hold | Fly camera (WASD + mouse-look) |
| Mouse wheel (flying) | Fly speed (multiplicative) |
| W / E / R (hovering) | Gizmo: translate / rotate / scale |
| F | Focus selected object |
| LMB click | Pick object (empty space deselects) |
| Ctrl+S | Save scene · Ctrl+Z/Ctrl+Y undo/redo · Del delete selection |

## Demo scene bindings

| Key | Action |
|---|---|
| 1 / 2 / 3 / 4 | Toggle ambient / directional / point / spot light |
| 5 | Toggle the deformation animation |
| 6 / 7 / 8 / 9 | Render mode: triangles / line strip / points / triangle fan |
